# Salud Flexible

This project use React + ViteJS

> [!TIP]
> We strongly recommend using `nvm` as a version manager

- Node.js (version >=18)

Use `yarn` to install all package:

```console
nvm use
yarn install
```

Finally run this to start project.

```console
yarn dev
```
